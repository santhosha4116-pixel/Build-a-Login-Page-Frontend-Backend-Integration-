🔐 Login Page – Frontend & Backend Integration

A full-stack login application built with React, Node.js, and Express.js. This project demonstrates how to create a modern login interface, validate user input, communicate with a backend API, and authenticate users using mock credentials.

🚀 Features
🔑 Modern and responsive login page
📧 Email and password input fields
✅ Frontend form validation
Empty field validation
Valid email format check
Password length validation
🌐 Backend API built with Express.js
🔄 Frontend and backend integration using Fetch/Axios
🔒 Mock authentication using static user credentials
⚠️ Error messages for invalid login attempts
🎉 Successful login redirects users to a Dashboard page
📱 Responsive design using Tailwind CSS / CSS

🎯 How It Works
Enter your email and password.
The application validates the input on the frontend.
Login credentials are sent to the Express backend using an API request.
The server checks the credentials against mock user data.
If the credentials are correct, the user is redirected to the Dashboard page.
If the credentials are incorrect, an appropriate error message is displayed.
